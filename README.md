
# SmallSCADA

SmallSCADA is a laboratory-scale supervisory control and data aquisition (SCADA) software aimed at automation of reactor hardware.



## About

Some information about the key sections of this repository.

### PFR

This folder contains the control code for Bronkhorst mass flow controllers (MFCs) and data collection code for an Agilent Gas Chromatograph. This code is used with a plug-flow reactor (PFR), but it can be used for any compatable equipment.

### Raman

This folder contains code for the data processing of Raman spectra.

### App

This folder contains the code for a graphical user-interface for use with the PFR code. This software is still in early developmental stages.


## Installation

To install this project, download or clone the project and run

```bash
  pip install requirements.txt
```
